# Pcap-X-Ray
Siber güvenlik analizleri için geliştirilmiş hafif siklet bir PCAP parser.   Bu araç, ağ paketlerini (PCAP/PCAPNG) doğrudan Ethernet katmanı seviyesinde okuyarak MAC adreslerini ve temel protokol verilerini ayıklar. Analiz sonuçlarını raporlara hızlıca aktarmak için kopyalanabilir bir arayüz sunar.
