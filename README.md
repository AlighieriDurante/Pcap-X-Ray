# Pcap-X-Ray

A lightweight PCAP parser developed for cybersecurity analysis.

This tool reads network packets (PCAP/PCAPNG) directly at the Ethernet layer to extract MAC addresses and basic protocol data. It features a user-friendly interface that allows for quick copying of analysis results into reports.

---

### Features
- **Fast Parsing:** Parses raw pcap data in seconds.
- **Tabular View:** Lists Source and Destination MAC addresses in a clear format.
- **Clipboard Support:** One-click "Copy to Clipboard" functionality for easy reporting.
- **Modern GUI:** Clean and responsive interface built with CustomTkinter.
